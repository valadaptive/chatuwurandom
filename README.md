A conversational AI interface powered by [uwurandom](https://github.com/valadaptive/uwurandom), the world's most advanced AI (artificial inanity) catgirl language model.

## Capabilities

- Unlimited context window (all input is discarded)
- All hallucinations
- Understands *all* languages equally well
- 100% open source and 100% local

## Limitations

- Cannot follow instructions
- Cannot answer questions
- Cannot not be a cat

## A Note on Safety

chatuwurandom has undergone no alignment training and exhibits no safety guardrails. It will not refuse any request, primarily because it does not understand them. The model may, however, occasionally eat all your doritos, which users should be aware of.
